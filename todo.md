Tambahan untuk sistem demo yang sudah berhasil dibangun sebelumnya:

Fokus: Memperbaiki format cetakan slip gaji agar sesuai contoh yang diberikan.

Instruksi:
1. **Format Slip Gaji**
   - Cetakan slip gaji harus menampilkan informasi dengan struktur rapi seperti contoh:
     - Header perusahaan (logo/nama perusahaan).
     - Periode gaji.
     - Data karyawan: Nama, NIP/NIPEG, Jabatan, Grade, Status Pegawai.
     - Bagian Penerimaan: Gaji Pokok, Tunjangan Pokok, Tunjangan Beras, Insentif Kinerja, Tunjangan Pajak, Uang Cuti Tahunan, dll.
     - Total Penerimaan.
     - Bagian Potongan: Biaya Jabatan, Premi JHT+JP, PPh 21 atas Gaji, PPh 21 atas Bonus, dll.
     - Total Potongan.
     - Gaji Bersih (Net Salary).
   - Gunakan tabel atau grid agar informasi penerimaan dan potongan jelas terbaca.
   - Pastikan angka ditampilkan dengan format mata uang Rupiah (Rp).

2. **UI/UX**
   - Slip gaji ditampilkan dalam halaman khusus dengan opsi cetak (print-friendly).
   - Gunakan layout sederhana, profesional, dan mudah dibaca.
   - Pastikan konsistensi font dan spacing agar mirip dengan slip gaji resmi.

3. **Data Dummy**
   - Gunakan data dummy (hardcoded/JSON) untuk simulasi slip gaji.
   - Pastikan contoh slip gaji yang ditampilkan sesuai struktur di atas.

4. **Dokumentasi**
   - Update README agar mencantumkan bahwa slip gaji sudah diperbaiki formatnya sesuai contoh.
   - Jelaskan bahwa data masih dummy, tetapi layout sudah siap untuk demo.
